### 3-1) 카테고리 생성 API (미구현)

설명: 새로운 카테고리(글을 쓸 수 있는 게시판)를 생성한다.  
경로: POST /api/circles/{circleId}/categories  
권한: 동아리 회장

```js
const request = {
  title: "Category Name", //카테고리 이름
  priority: 1, //정렬 우선순위 (nullable)
};

const response = {
  id: 1,
  title: "Category Name",
  priority: 1,

  createdAt: "",
  createdBy: {},
  lastModifiedAt: "",
  lastModifiedBy: {},
};
```
