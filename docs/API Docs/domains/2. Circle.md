### 2-1) 동아리 생성 API

설명: 새로운 동아리를 생성하는 API  
경로: POST /api/circles  
권한: 관리자

```js
const request = {
  name: "테스트 동아리", //동아리 이름
  description: "테스트 동아리", //동아리 설명
};

const response = {
  id: 1,
  name: "테스트 동아리",
  description: "테스트 동아리",
};
```

### 2-2) 동아리 팔로우 API

설명: 동아리를 팔로우하는 API  
경로: POST /api/circles  
권한: 로그인한 유저

```js
const request = null;

const response = {
  circle: {
    id: 1,
    name: "테스트 동아리",
    description: "테스트 동아리",
  },

  //열거형은 다음과 같이 가능한 모든 요소를 입력해줘야함
  type: "FOLLOWER / MEMBER",
};
```
